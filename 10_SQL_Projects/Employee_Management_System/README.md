# Employee Management System

## Project Overview

This project simulates an Employee Management System for a company.

The database stores information about:

- Employees
- Departments
- Jobs
- Salaries
- Projects

The project demonstrates SQL concepts such as:

- SQL Basics
- Aggregate Functions
- Joins
- Subqueries
- CTEs
- Window Functions
- Views
- Indexes

## Technologies Used

- MySQL
- MySQL Workbench
- SQL

## Skills Demonstrated

- Database Design
- Data Analysis
- Business Reporting
- Query Optimization
- SQL Problem Solving
